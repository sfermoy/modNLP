Concordance Meta-data facet distributions
This plugin can be used to see the number of lines in the current concordance which are associated with a particular piece of metadata, or to filter the concordance based on the metadata.
  • EXAMPLE
  • Select "author" form the dropdown.
  • Then expand/contract/slide the range view to relevant Authors.
  • Clicking on a bar will remove the lines associated with the attribute from the concordance list.
  • Once you are happy with your new concordance list the "Update Bars/Load concordance" button to view the metadata distributions for the filtered list
You may select multiple bars across all of the dropdown attributes. If the Mosaic plugin is active it will update as the concordance is filtered
